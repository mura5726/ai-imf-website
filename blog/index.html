解説 2026.06.24

FDE（Forward Deployed Engineer）とは何か

OpenAI／Anthropic／Palantir が採り入れる "顧客の現場に張り付くエンジニア" という役割の本質、なぜ AI コンサル時代に重要なのか、そして従来の SE／コンサルとの違いを解説します。
技術 2026.06.22

Claude Code でできる10のこと

ターミナル上で自律的にコードを書き、テストし、デプロイする ── 私たちが実務で使い倒している活用例を10通り。Web アプリの初稿、データ分析、社内ツール生成まで具体的に。
事例 2026.06.18

保険プランシミュレーターを作った話

家計のリスクを可視化する Web ツールを、なぜ／どうやって AI 駆動開発で作ったか。設計の意図、UI／UX で意識したこと、そして "Claude Code に何を任せ、何を任せなかったか"。
考察 2026.06.10

AIダッシュボードは "会話の触媒" である

"見るための画面" ではなく "経営層が議論を始められる画面" として、AIダッシュボードを設計する。Human in the Loop を前提にした UI 設計論。
